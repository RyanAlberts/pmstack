# Skill: Competitive Landscape Analysis

## Trigger
`/competitive [market or product]`

## Goal
Generate a structured, actionable competitive analysis for a given market or product.

## Instructions
When the user invokes this skill, you should:
1. Identify the top 3-5 competitors in the specified market.
2. For each competitor, analyze their:
   - Target audience
   - Core value proposition
   - Key strengths
   - Key weaknesses or gaps
3. Synthesize the findings into a "White Space" section: where is the opportunity for a new entrant or feature?
4. Format the output clearly using markdown tables or bullet points.
5. Save the output to `outputs/competitive-[market]-[date].md`.

## Tone
Objective, analytical, and direct. Avoid marketing fluff.
